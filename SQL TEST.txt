Data Script You Can Found Below

Task 1 : Design a Procedure "Get_display_output"
Task 2 :  which accepts one Input paremeter @display_id (id : 101 Or 102 or 103) as Given In Input_Master Table
Task 3 :  Proc gives output based on the rules given in Input_Master table for that display_id. 
          rule means  : from input_Master Table . Find Table_Name,ColumnList, And Filter_cols from @display_id 
          and result should be come from this dynamic respective table which is define in Input_Master

Note : You Need to Perform Dynamic Table Operation (single)


[Input_Master] table structure:
[display_id] - Display id which will be passed as a parameter
[Table_Name] - Table for which the output is required
[Select_Cols] - Columns required in the output
[Filter_Cols] - Columns on which filter must be applied


Plse use this script to create tables.
